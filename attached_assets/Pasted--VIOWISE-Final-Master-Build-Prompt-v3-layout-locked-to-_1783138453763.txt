Build a complete, premium, minimal-content clickable web prototype called **VIOWISE** — a platform where elderly mentors (60+) share life wisdom with young learners (18-35) through AI-facilitated video conversations. ## DESIGN SYSTEM

**Visual identity — NO PHOTOGRAPHY on light pages.** Instead: an abstract background icon layer — a sparse, oversized pattern of thin-line motifs (speech bubbles, linked circles, gentle leaf/branch lines) in violet at 4-6% opacity, consistent across all light pages, never behind text blocks (solid white cards always sit on top). People are represented by warm violet-tinted initial-avatars (e.g., a circle with "G" for Grace) — elegant, consistent, no stock photos. EXCEPTION: the video call screen uses realistic warm portrait imagery, because it depicts a live call.

**Colors:** page bg gradient 160deg #F7F5FB → #EFEAF9 (surfaces only) · cards white, border #E6E1F2, radius 16px, shadow rgba(83,64,155,0.10) · primary #53409B (hover #3D2F7A) · tints #EEEBF7 with border #C5BCDF · accent #8A7BC8 (badges only, never text) · green #16A34A confirmations only · red #DC2626 End call + report-danger only · dark call screen gradient #17141F → #1C1730, accent #A594E8.

**Typography:** Inter for UI; Fraunces ONLY for the landing headline and wisdom quotes (400 italic). Body 18px+, nothing below 16px anywhere. Buttons/inputs radius 12px, min 48px tall (primary CTAs 56-64px). Headline scale generous: landing H1 56px, page H1s 40px, eyebrow labels 16px uppercase letterspaced violet.

**Minimal content rule:** every page carries the fewest words that still instruct, reassure, or explain an AI action. No marketing filler. Trust/consent sentences are one line each and MUST NOT be cut.

**Accessibility (global, non-negotiable):** every icon labeled; toggles show On/Off text (18px); selection = border + checkmark + tint (never color alone); 3px violet focus rings; keyboard operable; aria-live on counters/confirmations; modals focus-trapped, Escape closes. Persistent **"Aa Accessibility"** button (48px, icon+text) in every nav: popover with Text size (Standard / Large 115% / Extra large 130% — via CSS variable, layouts must survive Extra large) and High contrast (On/Off — text #0F0D14, borders darker, gradient → solid). Persists across pages.
